Pré-requis
------------

Vous avez besoin :
* De java
* D'un éditeur
* [Typesafe activator > 1.2 ](http://downloads.typesafe.com/typesafe-activator/1.2.2/typesafe-activator-1.2.2.zip)

Démarrer
------------
Lancer <code>activator ui</code> depuis le répertoire racine du projet et laissez vous guider.
